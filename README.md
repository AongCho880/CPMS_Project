# CPMS_Project
This is the development branch

- git clone --branch branch-name https://github.com/username/repository.git

- create a .env file at server directory and copy
        jwtSecret = "secret";

------ Initialize Server --------
- cd .\server\
- npm install express
- npm start
- Ctrl + c to exit

------ Initialize admin --------
- cd .\admin\
- npm install
- npm start
- Ctrl + c to exit

------ Initialize shop --------
- cd .\shop\
- npm install
- npm start
- Ctrl + c to exit

----- Database -------
- edit db.js file as your own databse system required
- edit configdb.sh locate in bin directory as you end
